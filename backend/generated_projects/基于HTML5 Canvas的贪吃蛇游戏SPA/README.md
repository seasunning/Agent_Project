# 基于HTML5 Canvas的贪吃蛇游戏SPA

## 项目摘要
基于HTML5 Canvas的贪吃蛇游戏SPA，采用纯前端技术栈实现，包含游戏控制器、蛇管理、食物管理、渲染引擎、输入处理、分数管理和碰撞检测等模块。

## 技术栈
- HTML5
- CSS3
- JavaScript
- Canvas API

## 启动步骤
1. 确保已安装现代浏览器（如Chrome、Firefox）
2. 克隆或下载项目代码到本地
3. 使用浏览器直接打开frontend/index.html文件
4. 游戏将自动加载并显示开始界面
5. 按空格键或点击开始按钮启动游戏
6. 使用方向键控制蛇的移动
